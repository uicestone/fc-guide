# fc-guide

## Project setup
```
yarn install
```
then create a .env file from .env.example.

### Front-end development
```
yarn serve
```

### Server side development
```
yarn dev
```

### Compiles and minifies for production
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
